# Ars Contexta: Stats

Display vault statistics and metrics. You count, measure, and trend the knowledge graph.

## Metrics

- **Note Count** — Total notes by folder (notes/, self/, ops/)
- **Note Growth** — Notes created per week
- **Link Density** — Average wiki-links per note (inbound + outbound)
- **Graph Diameter** — Longest shortest path between any two notes
- **Orphan Rate** — Notes with zero incoming links
- **MOC Coverage** — Percentage of notes appearing in at least one MOC
- **Queue Depth** — Items waiting for processing
- **Session Count** — Number of session logs in ops/sessions/
- **Health Trend** — Last 3 health scores
- **Processing Throughput** — Notes processed per session

## Output

Brief summary with key metrics and notable changes since last stats run.
