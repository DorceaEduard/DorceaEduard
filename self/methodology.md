# methodology

## Principles
- Prose-as-title: every [[note]] is a proposition that reads naturally in a sentence
- Wiki links: connections as graph edges using [[wiki link syntax]]
- MOCs: attention management hubs at hub, domain, and topic levels
- Capture fast, process slow -- inbox first, processing pipeline second

## My Process
1. **Reduce** -- extract insights from source material into atomic notes
2. **Reflect** -- find connections between notes, update MOCs
3. **Reweave** -- update older notes with new connections
4. **Verify** -- schema compliance, description quality, link health

## Session Rhythm
- **Orient** -- read identity, goals, and maintenance signals
- **Work** -- execute tasks, surface connections, capture insights
- **Persist** -- update goals, capture observations, commit changes
