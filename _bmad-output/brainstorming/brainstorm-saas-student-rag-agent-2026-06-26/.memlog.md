- (technique) started How Might We
- (idea) Sistem care adauga knowledge prin cursuri inregistrate, YouTube, carti, PDF, prezentari — audio/video/doc — si face transcribe automat
- (decision) Fara accesibilitate - nu e focusul. Vom folosi Obsidian + ARS Context pentru second brain
- (idea) Plan de recuperare agentc: studentul absentat converseaza cu agentul si recupereaza conceptele esentiale in ordine inteligenta
- (direction) Finalizat explorarea How Might We - decizie: folosim ARS Contexta ca infrastructura de baza
- (technique) started Ship in 60 Minutes
- (idea) MVP nucleu: RAG + transcriere + chat agentic - primele 3 pentru prima versiune; restul vin dupa
- (technique) started Assumption Reversal
- (idea) Inversion: sistemul vine cu sursele - documente pre-încarcate pe Supabase, studentul doar se autentifica si intreaba pe o pagina web
- (decision) Toate documentele transcrise in format markdown - formatul canonic al knowledge base-ului
- (idea) Knowledge dublu: shared (facultate - cursuri generale pentru toti) + personal (note, inregistrari proprii)
- (technique) started Backcasting
- (idea) Backcasting moment final: studentul se lojeaza -> chat cu agent -> acceseaza sau incarca documente -> discuta pe vault general sau personal (doua vaulturi)
- (insight) Sinteza: SaaS agentic second brain cu RAG+transcriere+chat, doua vaulturi (shared+personal), input multi-format transcris in md, ARS Contexta + Obsidian, deploy pe Vercel/Render/Supabase/Resend, 90h/3persoane
