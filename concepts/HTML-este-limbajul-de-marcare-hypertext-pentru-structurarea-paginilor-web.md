# HTML este limbajul de marcare hypertext pentru structurarea paginilor web

HTML (HyperText Markup Language) reprezintă structura de bază a unui site web, oferindu-i organizare și formă. Stabilește modul în care sunt aranjate elementele afișate în pagină: text, imagini, formulare, butoane. În absența HTML-ului, paginile web ar conține doar text simplu.

**Hipertext** = text care conține linkuri către alte documente sau resurse, permițând navigarea între pagini web.
**Marcare** = etichetele (tag-urile) care definesc structura și formatarea conținutului (titluri, paragrafe, imagini, linkuri).
**Limbaj** = sistemul de reguli și sintaxă pentru a scrie și interpreta markup-ul în mod standardizat.
