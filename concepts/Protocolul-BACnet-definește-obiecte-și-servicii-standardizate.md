# Protocolul BACnet definește obiecte și servicii standardizate pentru controlul sistemelor de clădiri

BACnet (Building Automation Control Network) oferă obiecte cu proprietăți predefinite: AnalogInput (valoare, unitate, limite), BinaryOutput, Device, Calendar, Command, Schedule, Event Enrollment, File, Loop, Program. Serviciile includ citire/scriere proprietăți (ReadProperty, WriteProperty), descoperire dispozitive (Who-Is, I-Am), notificări COV, alarme și sincronizare timp. A fost creat special pentru control HVAC, iluminat, incendiu și securitate.
