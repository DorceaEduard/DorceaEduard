# Variabilele, operatorii și structurile de control stau la baza logicii JavaScript

**Tipuri primitive:** Number (întregi și virgulă mobilă), String (șiruri între ghilimele), Boolean (true/false), Undefined (variabilă neinițializată), Null (absența intenționată a valorii), Symbol (identificatori unici), BigInt (numere mari, sufix `n`).

**Tipuri de referință:** Object (`{ cheie: valoare }`), Array (`[elemente]`), Function.

**Operatorul `typeof`** returnează tipul unei variabile sub formă de string.

**Operatori:** aritmetici (`+`, `-`, `*`, `/`, `%`, `**`), atribuire (`=`, `+=`, `-=`), comparație (`==`, `===` — **strict**, `!=`, `!==`, `<`, `>`), logici (`&&`, `||`, `!`), ternar (`cond ? val1 : val2`), unari (`++`, `--`), pe biți, operatorul virgulă.

**Structuri de control condiționale:** `if...else if...else`, `switch`.

**Structuri repetitive:** `for`, `while`, `do...while`, `for...in` (proprietăți obiect), `for...of` (elemente colecție iterabilă).

**Gestionare excepții:** `try...catch...finally`, `throw`.

**Control salturi:** `break` (oprește bucla), `continue` (sare iterația curentă), `return` (oprește funcția).

**Scope (domeniu):** global, local (funcție), bloc (`let`/`const`), lexical (funcții imbricate), modul (export/import).
