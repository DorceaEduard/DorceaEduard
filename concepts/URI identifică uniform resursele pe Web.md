# URI identifică uniform resursele pe Web

Web-ul reprezintă un spațiu informațional alcătuit din resurse situate în locații numite noduri, utilizate prin intermediul unui sistem hypertext, folosind URI (Uniform Resource Identifier).

**URI = URL + URN**
- URL (Uniform Resource Locator) — localizează resursa
- URN (Uniform Resource Name) — numește resursa
