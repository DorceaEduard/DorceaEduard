# Nivelul rețea realizează rutarea pachetelor pe baza adreselor IP

Nivelul rețea (nivelul 3 OSI) are PDU-ul numit pachet. Funcții principale: încapsulare/decapsulare, rutare, adresare logică și verificarea integrității antetului. Adresarea IP este ierarhică (permite localizarea dispozitivelor în rețea), spre deosebire de adresarea plată (MAC). Sistemul este neorientat pe conexiune (packet-switched) — fiecare pachet poate urma o cale diferită. [[Nivelul legătură de date transmite cadre cu adrese MAC]] oferă servicii nivelului rețea.
