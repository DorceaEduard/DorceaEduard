# Web-ul a fost creat de Tim Berners-Lee la CERN în 1989

**Evoluția WWW:**
- **Anii '60–'80 (Pre-Web):** Apar primele rețele (ARPANET 1969), se dezvoltă TCP/IP, servicii FTP/Telnet/e-mail — informațiile erau greu de găsit
- **1989:** Tim Berners-Lee propune sistemul de gestionare a documentelor prin hipertext
- **1990:** Primul server web (httpd), primul browser (WorldWideWeb/Nexus), limbajul HTML, protocolul HTTP, prima pagină web
- **1991:** CERN publică tehnologia — Web-ul devine public
- **1993:** Browserul Mosaic face Web-ul accesibil publicului larg
