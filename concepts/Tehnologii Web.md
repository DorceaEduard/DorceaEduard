# Tehnologii Web

## Curs 1 — Introducere în World Wide Web (WWW)

### Definiții
- [[Internetul este o rețea globală de rețele interconectate prin TCP/IP]]
- [[WWW este un serviciu distribuit de informații hypermedia peste Internet]]
- [[URI identifică uniform resursele pe Web]]

### Tehnologii
- [[HTTP/HTTPS este protocolul de transfer al informației pe Web]]
- [[HTML este limbajul de marcare hypertext pentru structurarea paginilor web]]
- [[HTML a fost creat de Tim Berners-Lee la CERN în 1989]]
- [[URL este un subtip de URI care localizează resurse]]

### Curs 2-3 — HTML
- [[HTML este limbajul de marcare hypertext pentru structurarea paginilor web]]
- [[HTML a fost creat de Tim Berners-Lee la CERN în 1989]]
- [[Viitorul HTML include design mobil-first și integrare cu VR și IoT]]

### Curs 4 — CSS (proprietăți de formatare)
- [[CSS este limbajul de stilizare vizuală a paginilor web]]
- [[Proprietățile CSS de formatare vizuală controlează fundalul, textul, fonturile, linkurile, listele și tabelele]]

### Curs 5 — CSS Flexbox, Grid și Bootstrap Grid
- [[CSS Flexbox este un sistem de layout unidirecțional pentru distribuirea spațiului între elemente]]
- [[CSS Grid este un sistem de layout bidimensional cu rânduri, coloane și zone definite]]
- [[Bootstrap Grid este un sistem de layout responsive cu 12 coloane pentru design mobile-first]]

### Curs 6 — JavaScript (fundamente)
- [[JavaScript este un limbaj de programare interpretat pentru interactivitatea paginilor web]]
- [[Variabilele, operatorii și structurile de control stau la baza logicii JavaScript]]

### Curs 7 — JavaScript (DOM, Evenimente, Asincronism)
- [[DOM permite manipularea dinamică a elementelor și evenimentelor paginii web]]
- [[Asincronismul în JavaScript folosește Promises și Fetch API pentru operații neblocante]]

### Curs 8-9 — PHP (fundamente)
- [[PHP este un limbaj de scripting server-side pentru generarea paginilor web dinamice]]
- [[PHP suportă tipuri de date dinamice, operatori și funcții pentru prelucrarea datelor]]
- [[Funcțiile PHP permit organizarea modulară a codului cu parametri tipați]]

### Curs 10-11 — PHP (șiruri și mbstring)
- [[PHP manipulează șirurile de caractere cu patru moduri de delimitare și funcții specifice]]
- [[PHP necesită extensia mbstring pentru manipularea corectă a caracterelor UTF-8]]

### Istoric
- [[Web-ul a fost creat de Tim Berners-Lee la CERN în 1989]]
- [[Web 1.0 a însemnat pagini statice și conținut informativ]]
- [[Web 2.0 a adus conținut generat de utilizatori și aplicații interactive]]
- [[Web modern include AI, cloud, API-uri REST și aplicații SPA]]
