# Protocoalele de comunicare Lonworks, BACnet, KNX și OPC standardizează automatizarea clădirilor

Lonworks (Echelon) este protocolul peer-to-peer cel mai utilizat în BMS, cu suport pentru 32385 dispozitive în 255 subrețele. BACnet este singurul protocol care include toate mediile de transmisie, cu obiecte predefinite care scurtează proiectarea. KNX (din EIB, BatiBUS, EHS) oferă trei moduri de configurare (S, E, A). OPC (OLE for Process Control) permite accesul aplicațiilor Windows la date de la echipamente industriale.
