# Schema bloc a unui SRA conține regulator, element de execuție și reacție

x_i → [E] → [EC] → x_a (eroare) → [R] → x_c (comandă) → [EE] → x_m (execuție) → [IA] → x_e (ieșire). Reacția x_r se întoarce la EC. Funcționare: perturbația → x_e scade → x_r scade → x_a crește → R comandă → EE acționează → x_e revine la valoarea prescrisă.
