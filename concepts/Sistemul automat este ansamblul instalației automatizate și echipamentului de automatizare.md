# Sistemul automat este ansamblul instalației automatizate și echipamentului de automatizare

Instalația tehnologică (IT) este obiectul automatizării. Instalația automatizată (IA) = IT privită ca obiect al automatizării. SA realizează fără participarea omului funcții de: comandă, control, reglare, protecție sau optimizare automată.
