# Web modern include AI, cloud, API-uri REST și aplicații SPA

Web 3.0 / Semantic Web (2010-prezent). Caracteristici: aplicații cloud, AI și personalizare, API-uri REST, aplicații SPA (React, Angular, Vue), PWA, IoT, tehnologii blockchain.
