# Web 2.0 a adus conținut generat de utilizatori și aplicații interactive

Perioada 2000–2010. Caracteristici: conținut generat de utilizatori, aplicații web dinamice, interactivitate ridicată. Apar rețele sociale, bloguri, YouTube, AJAX, JavaScript avansat. Web-ul devine platformă completă de aplicații, nu doar documente.
