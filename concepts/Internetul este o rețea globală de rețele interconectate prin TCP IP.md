# Internetul este o rețea globală de rețele interconectate prin TCP/IP

Termenul Internet provine din "interconnected" și "network". Reprezintă o rețea globală unificată de calculatoare și dispozitive, fiecare având o adresă unică, interconectate prin protocoalele TCP/IP (stiva de protocoale Transmission Control Protocol și Internet Protocol).

**Servicii care funcționează pe Internet:**
- Poștă electronică (e-mail)
- Transfer de fișiere (FTP)
- Conectare la distanță (Telnet)
- World Wide Web (WWW)

**Important:** Internet ≠ WWW. Internetul este infrastructura fizică, Web-ul este un serviciu care rulează peste Internet.
