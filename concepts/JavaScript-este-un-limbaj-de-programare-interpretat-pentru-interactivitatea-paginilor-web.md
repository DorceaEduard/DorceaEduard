# JavaScript este un limbaj de programare interpretat pentru interactivitatea paginilor web

JavaScript este un limbaj **interpretat** (execută cod linie cu linie), **single-threaded**, **dynamic typing**, **event-driven** și **non-blocking**.

**Domenii de utilizare:** frontend web (interacțiune cu [[DOM permite manipularea dinamică a elementelor și evenimentelor paginii web]]), backend cu Node.js, aplicații mobile (React Native), IoT, jocuri și aplicații desktop.

**Includerea în HTML:**
- **Inline:** prin atribute `onclick`, `onmouseover`
- **Intern:** în tag-ul `<script>` din `<head>` sau `<body>`
- **Extern:** fișier `.js` legat prin `<script src="file.js">` (recomandat)

**Încărcare:** `async` (descărcare asincronă, executare imediată) și `defer` (executare după parsarea HTML-ului).

**Variabile:** `var` (function-scoped, hoisting), `let` (block-scoped, poate fi actualizată), `const` (block-scoped, nu poate fi reatribuită).
