# Clasificarea rețelelor se face după aria geografică

| Tip | Arie |
|-----|------|
| BAN | Body Area Network — imediata apropiere a utilizatorului |
| PAN | Personal Area Network |
| LAN | Local Area Network — arie restrânsă (locuință, birou) |
| CAN | Campus Area Network — cuprinde mai multe LAN-uri |
| MAN | Metropolitan Area Network |
| WAN | Wide Area Network |
