# HTML a fost creat de Tim Berners-Lee la CERN în 1989 și a evoluat prin versiuni majore

Prima pagină web a conținut text simplu și hyperlinkuri. Versiunile HTML:
- **HTML 1.0 (1993)** — prima versiune, funcții limitate
- **HTML 2 (1995)** — tabele și imagini încorporate
- **HTML 3 (1997)** — cadre, foi de stil, interactivitate cu JavaScript
- **HTML 4 (1997/1999)** — suport multimedia, continuarea progreselor
- **HTML 5 (2014)** — elemente semantice, suport audio/video nativ, controale de formular îmbunătățite
- **HTML 5.1+** — rafinări de performanță, securitate, instrumente pentru aplicații web moderne
