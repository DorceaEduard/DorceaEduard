# Software-ul ETS este instrumentul standard de configurare și programare a sistemelor KNX

ETS permite planificarea și proiectarea arhitecturii clădirii (niveluri, spații, tablouri), repartizarea echipamentelor pe încăperi, definirea topologiei rețelei (arii, linii, alocare automată adrese fizice), crearea grupurilor de adrese pentru funcții specifice și programarea fiecărei instalații (iluminat, încălzire, jaluzele, stație meteo). Versiuni: ETS1 (1993), ETS2 (1996), ETS3 (2004), ETS4 (2010-prezent).
